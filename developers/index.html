Welcome to RaeedChat for Developers!

Explore RaeedChat!

RaeedChat API allows you to bring RaeedChat to the next level! Get ready to create amazing things! With our apis you can create bots, and build off of RaeedChat!

API Endpoints

https://api.raeedchat.com/api/v1/sendMessage - Sends a message with bot api early access only

https://api.raeedchat.com/api/v1/sendReply - Sends a embed with bot api early access only

https://api.raeedchat.com/api/v1/searchUser - Search for a user in RaeedChat

Bot API

To use the bot api please contact us at webdevraeed@gmail.com! API Documentation will be given along with token :).

User Search API

Lets take a look at a request to searchUser!

 curl -XPOST 'https://api.raeedchat.com/api/v1/searchUser' \
 -H "Content-Type: application/json" \
 -d {"id": "USER_ID_HERE"}

This is sending a post request to our api with "id" equal to "USER_ID_HERE" where you would put the actual id of the user you are looking for. The response is this if user is found:

 {
    "code": 200,
    "success": true,
    "user": {
      "badges": ["Badge1", "Badge2", "Badge3"],
      "name": "USERNAME",
      "id": "USER_ID"
    }
 }
Field Type Description
id String The id of the user to lookup
